ServerManager

A Discord bot that handles your server for you. Create organized communities with powerful templates.

✨ Features

🎯

Template System

Create server structures from JSON templates. Set up a complete server in seconds.

💬

Smart Welcome

Welcome new members automatically with customizable messages and embeds.

🎭

Custom Roles

Create custom roles with permissions from your templates automatically.

📝

Anonymous Messages

Send anonymous messages with the .say command (admin only).

💾

Persistent Storage

Settings saved to database. Welcome channels persist across restarts.

🔧

Easy to Use

Simple commands. No complex setup. Just !setup and you're done.

⚡ Commands

!setup [template] [--name "Server Name"]
Set up your server with a template. Requires admin permission.
!welcomechannel
Enable/disable welcome messages in the current channel.
.say <message>
Send an anonymous message. Command is deleted. Admin only.
!invite
Get the bot invite link with permissions info.
!help
View all available commands and features.
!ping
Check if the bot is online and responsive.
